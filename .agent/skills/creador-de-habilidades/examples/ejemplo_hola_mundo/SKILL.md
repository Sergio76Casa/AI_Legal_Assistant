# Ejemplo Hola Mundo

Esta habilidad sirve como ejemplo didáctico para desarrolladores de habilidades.

## Funcionamiento

1. El agente lee este archivo.
2. Si el usuario pide un saludo especial, el agente usa esta guía.

## Instrucciones

Saluda al usuario diciendo "¡Hola desde la nueva habilidad!" y ofrece ayuda con el desarrollo de habilidades.
